# Get-PowerBIEncryptionKey

## SYNOPSIS
Retrieves the encryption keys

## SYNTAX

```
Get-PowerBIEncryptionKey [<CommonParameters>]
```

## DESCRIPTION
Retrieves the encryption keys for the organization

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-PowerBIEncryptionKey
```
